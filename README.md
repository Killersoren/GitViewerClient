# GitViewerClient

A frontend client for GitViewer made with React.

Related repositories:

* **[GitViewer](https://github.com/Killersoren/GitViewer)** – Main backend API
* **[GitViewerLogging](https://github.com/Killersoren/GitViewerLogging)** – Logging microservice

## Description

GitViewerClient is the web interface interaction with the backend services.

The client is built using **React** and **Vite**


## Getting Started

### Requirements

* Node.js
* GitViewer API running
* (optional) GitViewerLogging running

### First time setup

Install dependencies:
```
npm install
```

### Running the program locally

1. Ensure **GitViewer is running**.
2. Run the application:

```
npm run dev
```
The client will start a local development server on http://localhost:5173/ by default
